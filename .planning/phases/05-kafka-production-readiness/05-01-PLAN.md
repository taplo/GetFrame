<design_decisions>
The following decisions are at the agent's discretion per Phase 5 context. These are the chosen approaches:

1. **Schema Registry (KAFKA-05)**: Avro + Confluent Schema Registry via REST API.
   - `apache-avro` crate for Avro binary serialization
   - `reqwest` for Schema Registry HTTP calls
   - Wire format: Confluent-standard (magic byte 0x00 + 4-byte network-order schema ID + Avro binary payload)
   - Schema registered once on KafkaProducer startup; schema ID cached for lifetime
   - Falls back to JSON serialization when `schema_registry_url` is not configured

2. **At-least-once (KAFKA-04)**: `acks=all` + `enable.idempotence=true` + `message.timeout.ms=30000`.
   - rdkafka internally retries indefinitely within message.timeout.ms when idempotence is enabled
   - Application-level retry loop (3 attempts) kept as safety net for non-retriable errors
   - `acks` default changed from `"1"` to `"all"`

3. **Configurable topics (KAFKA-06)**: Per-stream `KafkaConfig.topic` override + `partition_key_field`.
   - `partition_key_field` values: `"stream_id"`, `"frame_number"` (default), `"timestamp"`
   - `StreamConfig.kafka: Option<KafkaConfig>` already exists — wiring added in StreamManager
   - `publish_metadata` signature extended with `topic_override: Option<&str>` and `partition_key: &str`

4. **Retention policy (FRAME-04)**: Application-level periodic cleaner using S3 ListObjectsV2 + DeleteObjects.
   - Works with any S3-compatible store (MinIO, Ceph, AWS)
   - Configurable `retention_days: Option<u64>` on StorageConfig (None = disabled)
   - Runs every 60 minutes, deletes objects with `LastModified < now - retention_days`
   - Deletes in batches of 1000 via DeleteObjects

5. **Error handling**: Keep the current 3-retry structure (1s/2s/4s). With idempotence, most delivery failures are handled by rdkafka internally. App-level retries catch edge cases (auth failures, topic not found, network partitions at startup). Persistent failures log at ERROR level and increment `KAFKA_ERRORS` counter.
</design_decisions>

    ```rust
    use aws_sdk_s3::Client;
    use aws_sdk_s3::primitives::ByteStream;
    use anyhow::Result;
    use chrono::{Utc, Duration};
    use std::sync::Arc;
    use tracing;

    pub struct RetentionCleaner {
        client: Client,
        bucket: String,
        retention_days: u64,
    }

    impl RetentionCleaner {
        /// Creates a new retention cleaner.
        pub fn new(client: Client, bucket: String, retention_days: u64) -> Self {
            Self { client, bucket, retention_days }
        }

        /// Runs one cleanup pass: lists objects, deletes those older than retention_days.
        /// Returns the number of objects deleted.
        pub async fn run_once(&self) -> Result<usize> {
            let cutoff = Utc::now() - Duration::days(self.retention_days as i64);
            let mut deleted = 0usize;
            let mut continuation_token: Option<String> = None;

            loop {
                let mut req = self.client
                    .list_objects_v2()
                    .bucket(&self.bucket);

                if let Some(token) = &continuation_token {
                    req = req.continuation_token(token);
                }

                let resp = req.send().await?;
                let keys: Vec<String> = resp.contents()
                    .iter()
                    .filter_map(|obj| {
                        let key = obj.key()?;
                        let last_modified = obj.last_modified()?;
                        let lm = last_modified.as_secs_f64();
                        let lm_chrono = chrono::DateTime::from_timestamp(lm as i64, 0)
                            .unwrap_or_default();
                        if lm_chrono < cutoff {
                            Some(key.to_string())
                        } else {
                            None
                        }
                    })
                    .collect();

                if !keys.is_empty() {
                    // DeleteObjects supports up to 1000 keys per request
                    for chunk in keys.chunks(1000) {
                        let del = self.client
                            .delete_objects()
                            .bucket(&self.bucket)
                            .delete(
                                aws_sdk_s3::types::Delete::builder()
                                    .set_objects(Some(
                                        chunk.iter().map(|k| {
                                            aws_sdk_s3::types::ObjectIdentifier::builder()
                                                .key(k)
                                                .build()
                                        }).collect()
                                    ))
                                    .build()
                            )
                            .send()
                            .await?;

                        let count = del.errors().len();
                        if count > 0 {
                            for err in del.errors() {
                                tracing::warn!(
                                    key = ?err.key(),
                                    code = ?err.code(),
                                    message = ?err.message(),
                                    "Retention cleanup failed to delete object"
                                );
                            }
                        }
                        deleted += chunk.len() - count;
                    }
                }

                match resp.continuation_token() {
                    Some(t) if !t.is_empty() => continuation_token = Some(t.to_string()),
                    _ => break,
                }
            }

            tracing::info!(
                bucket = %self.bucket,
                retention_days = self.retention_days,
                deleted_objects = deleted,
                "Retention cleanup pass completed"
            );
            Ok(deleted)
        }

        /// Spawns a periodic retention task. Runs `run_once` every `interval`.
        /// Logs errors but does not crash the task.
        pub fn start_periodic(self, interval: std::time::Duration) -> tokio::task::JoinHandle<()> {
            tokio::spawn(async move {
                let mut ticker = tokio::time::interval(interval);
                ticker.tick().await; // skip first immediate tick
                loop {
                    ticker.tick().await;
                    if let Err(e) = self.run_once().await {
                        tracing::error!(
                            error = %e,
                            "Retention cleaner pass failed"
                        );
                    }
                }
            })
        }
    }
    ```

    ```rust
    use anyhow::{Result, Context};
    use serde::{Deserialize, Serialize};

    #[derive(Debug, Clone)]
    pub struct SchemaRegistryClient {
        client: reqwest::Client,
        base_url: String,
        subject: String,
    }

    #[derive(Serialize)]
    struct RegisterRequest {
        schema: String,
        schema_type: String,
    }

    #[derive(Deserialize)]
    struct RegisterResponse {
        id: u32,
    }

    #[derive(Deserialize)]
    struct IdResponse {
        id: u32,
    }

    impl SchemaRegistryClient {
        /// Creates a new client for the given Schema Registry base URL.
        /// `subject` is the subject name under which the schema is registered
        /// (Confluent convention: "<name>-value").
        pub fn new(base_url: &str, subject: &str) -> Self {
            Self {
                client: reqwest::Client::builder()
                    .timeout(std::time::Duration::from_secs(10))
                    .build()
                    .expect("reqwest Client::builder() always succeeds"),
                base_url: base_url.trim_end_matches('/').to_string(),
                subject: subject.to_string(),
            }
        }

        /// Registers the given Avro schema with the Schema Registry.
        /// Returns the numeric schema ID assigned by the registry.
        /// If the schema already exists, returns the existing ID (idempotent).
        pub async fn register_schema(&self, schema_json: &str) -> Result<u32> {
            let url = format!("{}/subjects/{}/versions", self.base_url, self.subject);
            let body = RegisterRequest {
                schema: schema_json.to_string(),
                schema_type: "AVRO".into(),
            };

            let resp = self.client
                .post(&url)
                .json(&body)
                .send()
                .await
                .context("Failed to send register-schema request to Schema Registry")?;

            if resp.status().is_success() {
                let reg: RegisterResponse = resp
                    .json()
                    .await
                    .context("Failed to parse Schema Registry register response")?;
                Ok(reg.id)
            } else {
                // Schema Registry returns 409 Conflict if schema already exists
                // We then need to query for the existing schema ID
                if resp.status().as_u16() == 409 {
                    self.get_schema_id(schema_json).await
                } else {
                    let status = resp.status();
                    let text = resp.text().await.unwrap_or_default();
                    anyhow::bail!(
                        "Schema Registry register failed (HTTP {}): {}",
                        status, text
                    );
                }
            }
        }

        /// Queries the Schema Registry for an existing schema matching the given schema JSON,
        /// returning its ID.
        async fn get_schema_id(&self, schema_json: &str) -> Result<u32> {
            let url = format!("{}/subjects/{}/versions/latest", self.base_url, self.subject);
            // First try the latest version
            let resp = self.client
                .get(&url)
                .send()
                .await
                .context("Failed to query Schema Registry for latest schema")?;

            if resp.status().is_success() {
                let id_resp: IdResponse = resp
                    .json()
                    .await
                    .context("Failed to parse Schema Registry id response")?;
                return Ok(id_resp.id);
            }

            // If that fails, try posting with a verify-only call
            let url = format!("{}/subjects/{}", self.base_url, self.subject);
            let body = RegisterRequest {
                schema: schema_json.to_string(),
                schema_type: "AVRO".into(),
            };
            let resp = self.client
                .post(&url)
                .json(&body)
                .send()
                .await
                .context("Failed to look up existing schema in Schema Registry")?;

            if resp.status().is_success() {
                let id_resp: IdResponse = resp
                    .json()
                    .await
                    .context("Failed to parse Schema Registry lookup response")?;
                Ok(id_resp.id)
            } else {
                let text = resp.text().await.unwrap_or_default();
                anyhow::bail!(
                    "Schema Registry lookup failed (HTTP {}): {}",
                    resp.status(), text
                );
            }
        }
    }
    ```

        /// Publishes frame metadata to Kafka.
        ///
        /// - `topic_override`: If Some, use this topic instead of the default.
        /// - `partition_key`: The Kafka message key for partitioning.
        ///
        /// When Schema Registry is configured (KAFKA-05), serializes as Avro binary
        /// with Confluent wire format (0x00 + 4-byte schema ID + Avro payload).
        /// Otherwise falls back to JSON for backward compatibility.
        ///
        /// At-least-once delivery (KAFKA-04) is achieved via:
        /// - `acks=all` + `enable.idempotence=true` in producer config
        /// - Application-level retry loop for errors rdkafka cannot retry internally
        pub async fn publish_metadata(
            &self,
            frame: &ExtractedFrame,
            storage_url: &str,
            storage_bucket: &str,
            storage_key: &str,
            topic_override: Option<&str>,   // NEW per KAFKA-06
            partition_key: &str,            // NEW per KAFKA-06
        ) -> Result<()> {
            let metadata = FrameMetadata {
                stream_id: frame.stream_id.to_string(),
                source_type: "stream".to_string(),
                timestamp: chrono::Utc::now().to_rfc3339(),
                frame_number: frame.frame_number,
                rule_trigger: frame.rule_trigger.clone(),
                pts: frame.pts,
                storage_url: storage_url.to_string(),
                storage_bucket: storage_bucket.to_string(),
                storage_key: storage_key.to_string(),
                jpeg_size_bytes: frame.jpeg_bytes.len() as u64,
                jpeg_width: frame.width,
                jpeg_height: frame.height,
            };

            // Serialize — Avro or JSON per KAFKA-05 design decision
            let payload_bytes: Vec<u8> = if let Some(schema_id) = self.schema_id {
                self.serialize_avro(&metadata, schema_id)?
            } else {
                serde_json::to_string(&metadata)?.into_bytes()
            };

            let effective_topic = topic_override.unwrap_or(&self.topic);
            let key_str = partition_key.to_string();

            let mut last_error = None;
            let retry_delays = [
                Duration::from_secs(1),
                Duration::from_secs(2),
                Duration::from_secs(4),
            ];

            for (attempt, delay) in retry_delays.iter().enumerate() {
                let headers = OwnedHeaders::new()
                    .insert(Header { key: "stream_id", value: Some(&key_str) })
                    .insert(Header { key: "source_type", value: Some("stream") })
                    .insert(Header { key: "content_type", value: Some(
                        if self.schema_id.is_some() { "avro/binary" } else { "application/json" }
                    )});

                let record = FutureRecord::to(effective_topic)
                    .key(&key_str)
                    .payload(&payload_bytes)
                    .headers(headers);

                match self.producer.send(record, *delay).await {
                    Ok(delivery) => {
                        tracing::info!(
                            stream_id = %frame.stream_id,
                            frame_number = frame.frame_number,
                            topic = %effective_topic,
                            partition = delivery.partition,
                            offset = delivery.offset,
                            attempt = attempt + 1,
                            "Frame metadata published to Kafka"
                        );
                        return Ok(());
                    }
                    Err((e, _)) => {
                        tracing::warn!(
                            stream_id = %frame.stream_id,
                            frame_number = frame.frame_number,
                            topic = %effective_topic,
                            attempt = attempt + 1,
                            max_retries = retry_delays.len(),
                            error = %e,
                            "Kafka publish failed, retrying"
                        );
                        last_error = Some(e);
                        tokio::time::sleep(*delay).await;
                    }
                }
            }

            Err(anyhow::anyhow!(
                "Kafka publish failed after {} retries: {:?}",
                retry_delays.len(), last_error
            ))
        }

        /// Serializes FrameMetadata using Avro with Confluent wire format.
        /// Wire format: [magic byte 0x00] [4-byte big-endian schema ID] [Avro binary payload]
        fn serialize_avro(&self, metadata: &FrameMetadata, schema_id: u32) -> Result<Vec<u8>> {
            let value = frame_metadata_to_avro_value(metadata);
            let mut writer = apache_avro::Writer::new(&SCHEMA, Vec::new());
            writer.append(value)?;
            let avro_payload = writer.into_inner()?;

            // Prepend Confluent wire format headers
            let mut wire_payload = Vec::with_capacity(5 + avro_payload.len());
            wire_payload.push(0x00); // magic byte
            wire_payload.extend_from_slice(&schema_id.to_be_bytes()); // 4-byte schema ID
            wire_payload.extend_from_slice(&avro_payload);

            Ok(wire_payload)
        }
    }
